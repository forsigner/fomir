## Installation

#### Using Yarn:

```bash
yarn add fomir fomir-react
```

#### Using NPM:

```bash
npm i fomir fomir-react
```

## Basic usage

By default, fomir does not have any built-in ui library, here we use `fomir-unstyled` plugin:

```bash
npm i fomir-unstyled
```

```js
import { Fomir } from 'fomir'
import { FomirUnstyled } from 'fomir-unstyled'

Fomir.use(FomirUnstyled)
```

Then, use `useForm` api to create a form:

```jsx live=true
const BasicForm = () => {
  const form = useForm({
    onSubmit(values) {
      alert(JSON.stringify(values, null, 2))
      console.log('values', values)
    },

    children: [
      {
        label: 'First Name',
        name: 'firstName',
        component: 'Input',
        value: '',
      },
      {
        label: 'Last Name',
        name: 'lastName',
        component: 'Input',
        value: '',
      },
      {
        component: 'Submit',
        text: 'submit',
      },
    ],
  })

  return <Form form={form} />
}

render(<BasicForm />)
```
